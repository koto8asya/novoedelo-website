## Экран 1: Главная страница (/)

**Блоки:**
- Header: логотип + навигация + кнопки «Войти» / «Оставить заявку»
- Hero-секция: заголовок + CTA
- Блок услуг: 6 карточек (смартфоны, ноутбуки, планшеты, ТВ, консоли, техника)
- Преимущества: 4 иконки (гарантия, диагностика, мастера, скорость)
- Отзывы: карусель
- Footer: контакты + часы работы + соцсети

## Экран 2: Форма создания заявки (/cabinet/new-order)

**3-шаговый мастер:**
- Шаг 1 «Устройство»: тип → марка → описание неисправности
- Шаг 2 «Дата и время»: интерактивный календарь + слоты
- Шаг 3 «Подтверждение»: сводка + контакты + кнопка «Отправить»

## Экран 3: Административная панель (/admin/orders)

**Компоненты:**
- Боковая навигация (Sidebar)
- Панель фильтров: поиск + статус + дата
- Таблица заявок: № / клиент / устройство / мастер / дата / статус / действия
- Детальная боковая панель (Drawer) при клике на заявку

## User Flow: «Клиент оставляет заявку»

1. Главная → кнопка «Оставить заявку»
2. Страница авторизации → вход/регистрация
3. Личный кабинет → «Новая заявка»
4. Шаг 1: выбор устройства
5. Шаг 2: выбор даты
6. Шаг 3: подтверждение → «Отправить»
7. Экран успеха с номером заявки
8. Раздел «Мои заявки» — новая заявка со статусом «Новая»
