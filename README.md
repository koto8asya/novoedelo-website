## 📋 О проекте

Данный репозиторий содержит все наработки курсовой работы по разработке концепции и прототипирования веб-ресурса для **ООО «Новое дело»** — сервисного центра по ремонту бытовой электроники (г. Краснодар, ул. Красная, д. 47).

**Цель проекта:** Создание веб-приложения для автоматизации приёма заявок, отслеживания статуса ремонта и управления внутренними процессами сервисного центра.

---

## 📁 Структура репозитория

| Директория | Содержимое |
|---|---|
| `docs/bpmn/` | BPMN-диаграммы AS IS и TO BE |
| `docs/analysis/` | Аналитические документы, требования заказчика |
| `docs/sitemap/` | Иерархическая структура веб-приложения |
| `prototypes/` | Экспортированные материалы прототипа из Figma |
| `reports/` | Отчёт юзабилити-тестирования |
| `src/` | Заготовка структуры исходного кода |
