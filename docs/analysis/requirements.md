# Требования заказчика — ООО «Новое дело»

## Сведения об организации

- **Полное наименование:** Общество с ограниченной ответственностью «Новое дело»
- **ОГРН:** 1182375041234 | **ИНН:** 2308215678
- **Адрес:** г. Краснодар, ул. Красная, д. 47, офис 3
- **Сфера деятельности:** Ремонт и техническое обслуживание бытовой электроники (ОКВЭД 95.11, 95.12, 95.21)
- **Год основания:** 2018 | **Штат:** 12 сотрудников (8 мастеров)

## Выявленные проблемы (интервью с директором Кузнецовым А.В.)

1. Отсутствие онлайн-записи — клиенты обращаются только по телефону или лично
2. Ручное ведение базы заявок в Excel — риск потери данных
3. Нет системы автоматических уведомлений клиентов
4. Отсутствие публичного каталога услуг с ценами
5. Нет возможности оставить отзыв онлайн

## Требования к веб-ресурсу

| № | Категория | Требование |
|---|---|---|
| 1 | Запись клиентов | Онлайн-форма заявки с выбором услуги и даты |
| 2 | Статус ремонта | Личный кабинет клиента с отслеживанием заявки |
| 3 | Каталог услуг | Публичная страница с перечнем услуг и ценами |
| 4 | Администрирование | Панель управления заявками для сотрудников |
| 5 | Обратная связь | Форма вопросов/отзывов, онлайн-чат |
| 6 | Уведомления | SMS/Email-уведомления об изменении статуса |
